## About

Click and drag across the grid to draw, just like an Etch-a-Sketch toy. The grid size is adjustable — the more cells, the finer the resolution.

## Features

- Hover/drag to draw on a dynamic grid
- Adjustable grid size (user sets the number of cells per side)
- Grid resets when a new size is chosen
- Built with pure HTML, CSS, and JavaScript — no frameworks or libraries

## How to Run

1. Clone the repo:
```bash
   git clone https://github.com/KonstantinNovakovic/etch-a-sketch.git
```
2. Open `index.html` in a browser.

No build step or dependencies needed.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript (DOM manipulation, event listeners)
